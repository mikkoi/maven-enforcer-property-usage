Maven Enforcer Plugin Rule: Property Usage

Maven Enforcer Plugin provides goals to control certain environmental constraints such as Maven version, JDK version and OS family along with many more standard rules and user created rules.

property-usage-rule is a user created custom rule to verify the usage of defined properties in the source files of the build.

It often happens during development that properties are defined and used, and then taken out of use but definition is forgotten; it gets “left behind” in the resources/*.properties file. This plugin checks that all the defined properties are used.

  • For a quick adoption of the Enforcer rule, check the Quickstart page.
  • To know more about the rule’s configuration and parameters check the Usage page.
  • Released builds are available from Maven Central.
